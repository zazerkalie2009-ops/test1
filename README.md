# Меганавты — интерактивный прототип

Мобильный UI-прототип игры внутри приложения МегаФон. Он отражает основной игровой цикл из концепта: сбор кристаллов с лун, зарядка энергией, ежедневные задания, колесо удачи и карточки розыгрышей.

## Запуск

Откройте `index.html` в браузере — сборка и зависимости не требуются.

## Реализовано

- главная орбитальная сцена с планетой Нави и лунами;
- сбор готовых кристаллов и общий сбор;
- расход энергии на зарядку;
- навигация по заданиям, колесу и призам;
- интерактивное колесо с выдачей награды.
